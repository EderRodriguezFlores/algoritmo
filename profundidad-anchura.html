Algoritmos de Búsqueda en Profundidad y Anchura

Búsqueda en Profundidad (DFS)

La búsqueda en profundidad (DFS, por sus siglas en inglés) es un algoritmo para recorrer o buscar en árboles o grafos. Comienza en el nodo raíz y explora lo más profundo posible a lo largo de cada rama antes de retroceder. Es útil para encontrar un camino entre dos nodos o para recorrer un grafo.

¿Para Qué Sirve DFS?

  • Exploración de Grafos: DFS se utiliza para explorar todos los nodos y aristas en un grafo.
  • Resolución de Problemas: Se aplica en problemas como el juego de laberintos, búsqueda de rutas y resolución de rompecabezas.
  • Verificación de Conectividad: Determina si un grafo es conexo o no.

Ejemplo de DFS

Implementación en Código (Python): def dfs(graph, start, visited=None):
if visited is None:
visited = set()
visited.add(start)
for next in graph[start] - visited:
dfs(graph, next, visited)
return visited

# Ejemplo de uso
graph = {'A': {'B', 'C'}, 'B': {'A', 'D', 'E'}, 'C': {'A', 'F'}, 'D': {'B'}, 'E': {'B', 'F'}, 'F': {'C', 'E'}}
print("DFS traversal starting from 'A':", dfs(graph, 'A'))

Búsqueda en Anchura (BFS)

La búsqueda en anchura (BFS, por sus siglas en inglés) es un algoritmo para recorrer o buscar en árboles o grafos. Comienza en el nodo raíz y explora todos los nodos vecinos a nivel actual antes de pasar al siguiente nivel. Es útil para encontrar el camino más corto en grafos no ponderados.

¿Para Qué Sirve BFS?

  • Encontrar Caminos Cortos: BFS es ideal para encontrar la distancia mínima entre dos nodos en grafos no ponderados.
  • Exploración de Grafos: Se utiliza para explorar todos los nodos de manera sistemática.
  • Algoritmos de Redes: Se aplica en algoritmos de redes para encontrar rutas de comunicación.

Ejemplo de BFS

Implementación en Código (Python): from collections import deque

def bfs(graph, start):
visited = set()
queue = deque([start])
while queue:
vertex = queue.popleft()
if vertex not in visited:
visited.add(vertex)
queue.extend(graph[vertex] - visited)
return visited

# Ejemplo de uso
graph = {'A': {'B', 'C'}, 'B': {'A', 'D', 'E'}, 'C': {'A', 'F'}, 'D': {'B'}, 'E': {'B', 'F'}, 'F': {'C', 'E'}}
print("BFS traversal starting from 'A':", bfs(graph, 'A'))

Ventajas y Desventajas

DFS:

  • Ventajas: Puede encontrar una solución más rápidamente en algunos casos y es útil para problemas de recorrido completo.
  • Desventajas: No garantiza la solución más corta y puede usar mucha memoria en grafos grandes.

BFS:

  • Ventajas: Garantiza encontrar el camino más corto en grafos no ponderados y es útil para encontrar la distancia mínima.
  • Desventajas: Puede ser menos eficiente en términos de memoria en comparación con DFS.
